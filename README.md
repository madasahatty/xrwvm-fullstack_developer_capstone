# Best Cars Dealership Capstone Project

This project is a full-stack dealership review portal for Best Cars, a national car dealership with branches across the United States.

## Project Overview

The application allows users to view dealership locations, filter dealerships by state, view dealership details, and read customer reviews. Registered users can log in and submit reviews for dealerships. Admin users can manage car makes and models through the Django admin interface.

## Features

- Static About Us and Contact Us pages
- User registration, login, and logout
- Dealer listing page
- Dealer filtering by state
- Dealer detail pages with reviews
- Review submission for logged-in users
- Django admin support for managing car makes and models
- Backend API endpoints for dealerships, reviews, cars, and sentiment analysis
- GitHub Actions workflow for CI/CD
